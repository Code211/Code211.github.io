Workshops

Web Development 101

Description:
Beginner-friendly web development class introducing the fundamentals of CSS, HTML, and JavaScript in a simple, easy-to-follow format. No prior experience necessary — just bring your curiosity and creativity!

Timing: 9:30 AM - 10:00 AM

AI Hand Gesture Detection

Description:
In this introductory computer vision workshop, we will introduce how AI Computer Vision Models make predictions based on imputed images using training data. We’ll make a Hand Gesture Detection web app with TensorFlow.js, Firebase storage and APIs, and some JavaScript. No prior experience with TensorFlow.js/Firebase necessary! JavaScript experience is recommended but not required.

Timing: 12:30 PM - 1:00 PM

Computer 3D Graphics

Description:
In this workshop, you will learn the fundamental steps to writing your own 3D renderer using rasterization techniques. This includes perspective and rotation matrices, clipping, shading, and more. Through this, you will gain a better understanding of the math and techniques used in games and other software to optimize 3D rendering for real-time visuals and beyond.

Timing: 2:30 PM - 3:00 PM
